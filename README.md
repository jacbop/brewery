# brewery

```
git clone git@github.com:martylamb/nailgun.git
cd nailgun
make && sudo make install
cd ..
git clone git@github.com:jacbop/brewery.git
cd brewery
mvn install
./run
ng temp
```

delta zero - temp at which delta is "zeroed"

## vol

## strike water
grain temp, absorption

## efficiency
